# AI_Chatbot

## Project Description

This project is a simple AI Chatbot developed using Python. The chatbot can respond to predefined user queries based on intent matching. It supports multiple conversation intents such as greetings, help requests, name queries, weather queries, AI-related questions, and more.

## Technologies Used

* Python

* JSON
  
* NLTK

## Supported Intents

* Greeting (Hi, Hello, Hey)

* Name Query

* Age Query
  
* Thank You
  
* Help

* AI Information

* Project
  
* Goodbye

## How to Run

1. Open the project folder in VS Code.
   
2. Open the terminal.

3. Run the following command:

* python chatbot.py

4. Start chatting with the bot.

5. Type quit to exit the chatbot.
